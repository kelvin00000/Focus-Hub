# Focus Hub

Focus Hub is a centralized productivity and collaboration platform built to improve how university students study, communicate, and work on academic projects together.

The platform combines document management, real-time collaboration, intelligent study tools, and productivity tracking into a single workspace to reduce the friction caused by constantly switching between multiple applications.

## Overview

Modern students often rely on several disconnected tools for messaging, document sharing, research, revision, and project coordination. This fragmented workflow can reduce focus, organization, and overall productivity.

Focus Hub addresses this problem by providing a unified environment where students can:

- Collaborate on academic work in real time
- Communicate within study groups
- Access AI-assisted learning tools
- Manage and view study materials
- Track personal and group productivity

## Core Features

### Group Features
- In-app document viewing
- Real-time collaboration and messaging
- AI chat and web search integration
- Flashcard generation
- Quiz generation
- Collaboration analytics dashboard

### Personal Features
- In-app document viewing
- AI chat and web search integration
- Flashcard generation
- Quiz generation
- Study analytics dashboard
- Focus mode for distraction-free studying

## System Goals

- Remove app-switching during study sessions
- Improve collaboration efficiency among students
- Support smarter revision and learning workflows
- Encourage productivity and focused study habits
- Provide a scalable modern academic workspace
